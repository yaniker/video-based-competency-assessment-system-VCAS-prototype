FLS Skills Automated Scoring System

1. Purpose
The purpose of this prototype is to demonstrate the feasibility and functionality of the proposed automated FLS skills scoring system.

#########################################################################################################################################################################

2. Folder Structure
FLS Skills Automated Scoring System Prototype V0.1
│
├── Successful_UI_View.mp4
│           • Screen recording of the UI throughout the procedure.
│
├── Unsuccessful_UI_View.mp4
│           • Screen recording of the UI throughout the procedure.
│
└── Outcome_Detailed.png
            • Explanations of the UI results, detailing what the displayed results mean. (Same for both)


#########################################################################################################################################################################

3. User Instructions
    1. View the "Successful_UI_View.mp4" and "Unsuccessful_UI_View.mp4" for UI views of the procedures.
    2. Refer to the "Outcome_Detailed.png" for a comprehensive understanding of the results presented by the UI.
